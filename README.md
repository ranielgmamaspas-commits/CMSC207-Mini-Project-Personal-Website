# Rain's Personal Website
This project is a personal website created as part of the CMSC 207 course.

## Features
- Navigation menu with working anchor links
- About Me section with profile image
- Hobbies section with images
- Contact section with Gmail, LinkedIn, and GitHub links
- Fun Fact button using JavaScript
- Mobile responsive design using CSS media queries

## Project Structure
├── index.html
├── style.css
├── script.js
├── README.md
└── images/
    ├── profile.jpg
    ├── cooking.png
    ├── piano.png
    ├── dogs.png
    ├── gmail.png
    ├── linkedin.png
    └── github.png

## Sections
- **Home** – Welcome banner
- **About Me** – Short bio with a fun fact button
- **Hobbies & Interests** – Three hobby cards with images
- **Contact** – Gmail, LinkedIn, and GitHub links with icons

## How to Run
1. Open `index.html` in any web browser.
2. No server or installation needed — it runs entirely in the browser.
